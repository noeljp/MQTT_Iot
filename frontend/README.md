# APRU40 Frontend

React TypeScript frontend for the APRU40 IoT Management Platform.

## Features

- **Dashboard**: Real-time monitoring with key performance indicators
- **Devices Management**: View and manage all IoT devices (nodes and gateways)
- **Gateways Overview**: Monitor gateway status and connected nodes
- **Alerts Management**: View, acknowledge, and resolve security alerts
- **Real-time Updates**: WebSocket integration for live data updates
- **Material-UI Design**: Modern, responsive UI components
- **JWT Authentication**: Secure login and authorization

## Prerequisites

- Node.js 14+ 
- npm or yarn

## Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

The application will open at `http://localhost:3000`

## Authentication

Default demo credentials:
- Username: `admin`
- Password: `admin123`

## Technologies Used

- React 18, TypeScript, Material-UI, React Router, Axios, Socket.IO Client
